选股策略框架

政策双轮驱动 + 戴维斯双击
选股策略

自上而下定方向，自下而上选个股——用政策找方向，用财务找时机，用仓位控风险。

策略构建 2026年7月 适用市场：A股 / 港股 / 美股中概

一、策略核心逻辑

本策略构建在两个独立但可叠加的逻辑层之上，目标是筛选出同时具备"政策推动力"和"估值修复力"的标的——这种三重共振区间的标的，赔率最高、安全边际最大。

逻辑层一：供需双轮驱动

  • 供给端：政策限产、行业整合出清、产能优化
  • 需求端：政策刺激消费、以旧换新、扩大内需
  • 双击信号：量价齐升、格局改善、龙头集中度提升
  • 典型案例：供给侧改革后的煤炭/钢铁（2016-2018）

逻辑层二：戴维斯双击

  • 盈利上修：EPS 超预期增长，业绩持续兑现
  • 估值扩张：市场重估 PE/PB，预期差修复
  • 核心公式：股价涨幅 = EPS 增长 × PE 扩张倍数
  • 必要前提：当前估值处于历史低位 + 盈利拐点确认
策略精髓：先用政策窗口定位有供需格局改善潜力的行业（行业筛选），再在行业内部找到低估值 + 业绩拐点 + 催化剂的个股（公司筛选）。两者叠加区 = 最优击球区。
最优标的 = 供给端收缩 ∩ 需求端扩张 ∩ 低估值业绩拐点

二、戴维斯双击量化筛选标准

五个维度量化打分，五项均满足 → 纳入候选池；满足前4项（催化剂缺失）→ 观察池，等确认信号。

① 低估值底线

PE< 行业均值 × 0.7
PB< 1.5（周期）/ < 3（成长）
股价回撤距历史高点 ≥ 40%
EV/EBITDA低于同业中位数

② 业绩拐点信号

毛利率连续 2 季度回升
分析师预期连续上调 EPS 预测
净利润同比由负转正
经营现金流由负转正趋势

③ 盈利加速器

营收增速连续 2 个季度加速
ROE底部回升趋势确认
利润弹性净利润增速 > 营收增速
运营效率存货/应收增速 < 收入增速

④ 资产负债表健康

资产负债率< 60%（非金融企业）
有息负债/EBITDA< 3x
商誉/总资产< 10%
流动比率> 1.2（流动性安全）

⑤ 催化剂识别（触发买入时点）

政策催化剂新政策落地 / 补贴清单公布 / 行业标准升级 / 进口替代加速
业绩催化剂重大订单签订 / 产能达产释放 / 新产品放量
市场催化剂行业景气指数触底回升 / 机构持仓占比触底反转 / 分析师覆盖启动
估值催化剂同行业可比公司估值提升 / 公司回购 / 大股东增持

三、当前具备双轮驱动潜力的方向

声明：以下方向基于2026年宏观政策逻辑推导，仅供研究参考，不构成投资建议。具体标的需结合上述五项量化标准逐一筛选。

方向 供给端逻辑 需求端逻辑 双击潜力
家电 / 消费电子 落后产能出清，龙头集中度持续提升 以旧换新补贴持续，出口拉动 高
工程机械 海外产能布局完成，国内过剩整合 基建补短板 + 海外新兴市场需求 高
半导体（国产替代） 先进制程差异化，落后品类整合 国产化率提升 + AI 算力需求 中高
医疗器械 集采出清低质产品，格局改善 医保扩容，消费医疗复苏 中高
猪周期（养殖） 产能去化基本到位 消费回暖，猪价上行周期 高

四、排雷清单 — 这些标的不碰

以下特征的标的直接排除，不进入候选池：

  • 政策利好已被充分定价，PE 高于历史 80% 分位
  • 供给端没有实质性出清，行业 CR5 < 30%
  • 需求端依赖一次性补贴，无持续需求支撑
  • 资产负债率 > 70%，经营性现金流持续为负
  • 大股东持续减持 / 商誉超过净资产 30%
  • 近一年内发生过财务造假 / 审计意见非标
  • 市值 < 50 亿 RMB（流动性不足，机构不碰）
  • 日均换手率 < 0.3%（流动性枯竭风险）

五、仓位管理原则

标的类型 单票仓位上限 操作原则
三重共振（双轮+双击均满足） ≤ 15% 分批建仓，季报验证后加仓
双轮驱动但尚未双击 ≤ 8% 等业绩拐点确认后再加
纯主题 / 政策预期 ≤ 5% 轻仓博弈，快进快出
组合总仓位 ≤ 70% 预留 30% 现金应对市场波动
建仓节奏建议：第一笔建 30-40%（确认低估值+趋势），等季报/半年报验证业绩拐点后再加 30-40%（戴维斯双击确认），剩余仓位等催化剂事件触发（政策落地/订单公布）。

六、选股操作流程（5步法）

1
行业扫描 每周浏览国务院/发改委/工信部政策文件 → 锁定有实质性政策支持的行业方向
2
财务初筛 对锁定的行业，用五项量化标准跑全行业数据 → 筛选满足 ≥4 项的标的
3
深度尽调 阅读近 2 期季报/半年报 + 分析师报告 → 验证业绩拐点是否为结构性（而非一次性）
4
催化剂匹配 确认未来 1-3 个月有无明确的催化剂事件窗口 → 确定买入时点
5
仓位部署 按仓位管理原则分批建仓 → 设好止损线（-15%硬止损）→ 季报窗口再评估

七、主要风险与应对

风险类型 具体表现 应对措施
政策落地不及预期 文件发了，补贴资金没到位 持仓 < 15% 上限，不要等政策兑现才减持
业绩兑现延迟 拐点预期出现，但季报仍亏损 以季报/半年报为硬验证窗口，不确认不加仓
外部冲击 汇率剧烈波动 / 地缘冲突 / 流动性收紧 保留 30% 现金仓位，-15% 无条件止损
估值陷阱 低估值≠便宜，可能是价值毁灭 低估值必须搭配业绩拐点确认，缺一不可
风格轮动 市场偏好偏离价值/周期风格 不做短线博弈，坚守策略纪律
